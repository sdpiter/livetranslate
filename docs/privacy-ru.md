# LiveTranslate — Политика конфиденциальности
Последнее обновление: 2025‑MM‑DD

## Кратко
LiveTranslate обрабатывает звук и текст локально на вашем устройстве. Мы не отправляем аудио или переводы на серверы.

## Какие данные обрабатываются
- Аудио с микрофона и/или системный звук обрабатываются на устройстве для распознавания и перевода. Эти данные не передаются и не сохраняются после обработки.
- История субтитров хранится только в памяти текущей сессии.

## Какие данные собираем
Для диагностики и стабильности можем собирать:
- Данные о сбоях и производительности (модель устройства, версия ОС, версия приложения, стек ошибок).
- Технический идентификатор установки (не привязан к личности).
Мы не собираем аудио, распознанный текст и персональные данные.

## Как используем данные
- Диагностика, исправление ошибок, улучшение качества.

## Передача и хранение
- Диагностика может обрабатываться сервисом краш‑репортинга (например, Firebase Crashlytics) по защищённому соединению (TLS).

## Разрешения
- Микрофон — для распознавания речи.
- Показ поверх других приложений — для оверлея субтитров.
- Захват экрана/системного звука — только по вашему явному согласию, для субтитров к воспроизводимому контенту.  
Приложение не записывает телефонные звонки и не работает с защищённым DRM‑контентом.
